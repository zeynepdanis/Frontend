HTML Images

HTML Images Syntax

The HTML tag is used to embed an image in a web page. Images are not technically inserted into a web page; images are linked to web pages. The tag creates a holding space for the referenced image.

Images on Another Server/Website

To use an image from another server, you will need to save the image to your computer and upload it to your own server. You can also use an image on the web. You will need the full URL to the image, which includes the domain name.

HTML Image Maps

An image map allows you to link different parts of the image to different URLs. Use the tag to define an image map. Use the tag to define the clickable areas in the image map.

The HTML Element

The HTML element gives web developers more flexibility in specifying image resources. The element contains one or more elements, each referring to different images through the srcset attribute. This way the browser can choose the image that best fits the current view and/or device. Each element has a media attribute that defines when the image is the most suitable.
